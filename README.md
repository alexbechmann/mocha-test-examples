# Integration Testing with Mocha & Chai (NodeJS)

## Prequisites

- NodeJS
- Start function app (see ../README.md for instructions)

## Start Selenium drivers

Install selenium drivers (choco install selenium-all-drivers)

OR

```bash
docker-compose up  -d

```

## Run tests

```bash
npm install
npm test
```
